# 二、内部函数
1. ```help elfun```命令查看相关全部内部函数
2. ```help functionname```查看functionname函数的用法
3. ```rand()```产生[0,1]之间的随机数，```rng(k)```制定随机种子，```randn()```产生正态分布的随机数，等等
